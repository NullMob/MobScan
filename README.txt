# PortScan

## Sobre

PortScan é um programa de escaneamento de portas escrito em Python. Ele utiliza a biblioteca `socket` para verificar se uma porta em um host específico está aberta ou fechada.

## Funcionalidades

- Escaneamento de portas em um host específico
- Verificação se uma porta está aberta ou fechada

## Como usar

1. Clone o repositório para o seu ambiente local.
2. Ative o ambiente virtual (se você estiver usando um).
3. Execute o programa com o comando `python main.py`.

## Tecnologias

- Python
- Socket
